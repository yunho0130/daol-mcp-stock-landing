AI 에이전트가 주식 시세를 안전하게 불러오는 방식

MCP는 AI 모델이 증권 시세, 환율, 포트폴리오 시스템 같은 외부 도구를 표준 인터페이스로 호출하게 해주는 연결 규격입니다. 다올투자증권 컨셉 페이지는 이 구조를 활용해 실시간형 시세 조회와 투자상품 안내까지 하나의 경험으로 묶어 보여줍니다. 아래 상세 구현은 yfinance와 Python MCP SDK로 한국 주식 시세 MCP 서버를 만드는 절차를 반영했습니다.

How MCP Works

MCP는 AI와 금융 데이터 사이의 표준 커넥터입니다

모델이 임의로 웹을 긁는 대신, 허가된 MCP 서버가 데이터 소스와 인증, 호출 스키마, 응답 형식을 책임집니다. AI는 필요한 시점에 도구를 호출하고, 사용자는 출처와 시점을 확인할 수 있습니다.

1

도구 선언

MCP 서버가 `주식 시세 조회`, `환율 조회`, `상품 링크 조회` 같은 기능을 JSON 스키마로 공개합니다.

2

컨텍스트 호출

AI 에이전트는 질문 맥락에 맞춰 필요한 도구만 호출하고, 티커·종목코드·통화 같은 인자를 명확히 전달합니다.

3

검증 가능한 응답

응답에는 가격, 기준시각, 출처, 지연 여부를 포함해 사용자가 데이터 신뢰도를 판단할 수 있게 합니다.

System Overview

증권 서비스에 맞춘 MCP 시세 파이프라인

프론트엔드는 AI 에이전트와 대화형 UI를 제공하고, MCP 서버는 금융 API 접근 권한과 호출 제한을 관리합니다. 민감 정보는 모델이 아니라 서버 사이드 경계에서 다룹니다.

Implementation Detail

한국 주식 시세 MCP 서버 구현 상세

첨부된 korean-stock-mcp-builder 스킬 내용을 반영해, 한국 주식(KOSPI/KOSDAQ) 시세를 무료 데이터 소스인 yfinance와 Python MCP SDK로 제공하는 서버 구성을 설명합니다. API 키 없이 시작할 수 있고, Claude Code·Claude Desktop 같은 MCP 클라이언트에 등록해 사용할 수 있습니다.

Python MCP SDKyfinanceKOSPI/KOSDAQAPI Key 불필요stdio transport

get_quote

티커를 입력하면 현재가, 전일 대비 등락, 등락률, 통화, 기준 시각을 반환합니다. 예: 005930, 005930.KS, 035720.KQ

search_symbol

회사명이나 키워드로 Yahoo Finance 심볼 후보를 검색합니다. 예: 삼성전자, 카카오

get_history

기간·봉 간격 기준으로 과거 OHLCV 데이터를 반환합니다. 예: period=1mo, interval=1d

한국 시장 처리 규칙

  • 6자리 종목코드만 입력되면 KOSPI 접미사 .KS를 우선 시도합니다.
  • KOSPI 결과가 없으면 KOSDAQ 접미사 .KQ로 자동 재시도합니다.
  • 티커 정규화는 _resolve(ticker) 헬퍼 한 곳에서 처리해 중복 로직을 줄입니다.
  • 반환 값은 MCP 직렬화를 위해 문자열·float·int 등 JSON 호환 타입으로 변환합니다.

데이터 한계와 고지

  • Yahoo Finance 무료 데이터는 한국 시장 기준 약 15~20분 지연될 수 있습니다.
  • 완전한 실시간 체결가가 필요하면 한국투자증권(KIS) OpenAPI 등 증권사 API로 교체해야 합니다.
  • 응답에는 note 필드를 넣어 지연 가능성과 데이터 출처를 함께 안내합니다.
  • 없는 티커·네트워크 실패·빈 데이터는 조용히 무시하지 않고 error로 반환합니다.

구현 절차

프로젝트 스캐폴드uv init korean-stock-mcp 후 uv add "mcp[cli]" "yfinance>=0.2.40"로 의존성을 설치합니다.
서버 구현FastMCP("korean-stock")를 만들고 @mcp.tool()로 3개 툴을 등록합니다.
로컬 테스트uv run mcp dev server.py로 MCP Inspector를 열어 삼성전자·카카오·KOSDAQ 예시를 호출합니다.
클라이언트 등록프로젝트 루트의 .mcp.json 또는 Claude Desktop 설정에 절대경로 기반 stdio 서버로 등록합니다.
