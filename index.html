How MCP Works

MCP는 AI와 금융 데이터 사이의 표준 커넥터입니다

모델이 임의로 웹을 긁는 대신, 허가된 MCP 서버가 데이터 소스와 인증, 호출 스키마, 응답 형식을 책임집니다. AI는 필요한 시점에 도구를 호출하고, 사용자는 출처와 시점을 확인할 수 있습니다.

1

도구 선언

MCP 서버가 `주식 시세 조회`, `환율 조회`, `상품 링크 조회` 같은 기능을 JSON 스키마로 공개합니다.

2

컨텍스트 호출

AI 에이전트는 질문 맥락에 맞춰 필요한 도구만 호출하고, 티커·종목코드·통화 같은 인자를 명확히 전달합니다.

3

검증 가능한 응답

응답에는 가격, 기준시각, 출처, 지연 여부를 포함해 사용자가 데이터 신뢰도를 판단할 수 있게 합니다.

System Overview

증권 서비스에 맞춘 MCP 시세 파이프라인

프론트엔드는 AI 에이전트와 대화형 UI를 제공하고, MCP 서버는 금융 API 접근 권한과 호출 제한을 관리합니다. 민감 정보는 모델이 아니라 서버 사이드 경계에서 다룹니다.

Client
사용자 질의“삼성전자 현재가와 달러 환산 가격 보여줘”
AI Agent
도구 선택종목코드 005930, KOSPI, USD/KRW 환율 조회 필요성을 판단
MCP Server
권한 있는 호출시세 API와 환율 API를 표준 응답으로 정규화
Landing UI
